# GeoJSON-query

![geoqueryapp](img.png)

A useful web application to create GeoJSON format polygon for specific region on the map.

**myconstants.py** may look like:

    host = "localhost"
    appkey = "kakao-web-api"

Run Flask app by:

    python flaskapp.py
